Logo Description
This Logo of this website has the same colors as the website. In the center of the Logo, you can see தமிழ் written in white. This is how we write Tamil in our language. Round about there is a Thirukural, which is like a poem written.
அகர முதல எழுத்தெல்லாம் ஆதி பகவன் முதற்றே உலகு
{Akara Mudhala Ezhuththellaam Aadhi Pakavan Mudhatre Ulaku}
It’s the first poem in this literature books. The meaning of this poem is: "A" is the first and source of all the letters. Even so is God Primordial the first and source of all the world.
